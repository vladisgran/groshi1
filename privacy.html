Політика конфіденційності та захисту персональних даних

Ця Політика діє для всієї інформації, яку Виконавець (ФОП [Твоє ПІБ]) може отримати про Користувача під час використання сайту:

https://vladisgran.github.io/groshi1/

1. Які дані ми збираємо

1.1. Під час реєстрації на курс або здійснення оплати, Користувач надає наступні дані:

  • Прізвище, Ім'я, По батькові
  • Контактний номер телефону
  • Адреса електронної пошти (e-mail)
  • Нікнейм у месенджерах (Telegram тощо)

1.2. Сайт також може автоматично збирати технічні дані (IP-адреса, файли cookies, дані про браузер) для покращення роботи сервісу.

2. Мета збору даних

2.1. Персональні дані Користувача використовуються для:

  • Ідентифікації Замовника для надання доступу до Курсу
  • Здійснення зворотного зв'язку (консультації, сповіщення про оновлення)
  • Обробки платежів через платіжні системи
  • Надсилання маркетингових матеріалів та спеціальних пропозицій (від яких Користувач може відмовитися в будь-який момент)

3. Захист та передача даних

3.1. Виконавець вживає всіх необхідних організаційних та технічних заходів для захисту персональних даних від несанкціонованого доступу третіх осіб.

3.2. Передача даних третім особам здійснюється виключно:

  • Для забезпечення роботи платіжних сервісів (LiqPay, WayForPay тощо)
  • На вимогу законодавства України у встановленому законом порядку

4. Права користувача

4.1. Користувач має право:

  • Знати, які саме дані про нього зберігаються
  • Вимагати видалення своїх даних із бази Виконавця
  • Відкликати згоду на обробку персональних даних, надіславши запит Виконавцю

Для реалізації своїх прав надішліть запит на: support@vladisgran.com

5. Згода з умовами

5.1. Користувач підтверджує, що надання персональних даних під час оформлення замовлення є добровільною згодою на їх обробку згідно з цією Політикою.

5.2. Виконавець має право оновлювати Політику конфіденційності. Зміни вступають в силу з моменту їх публікації на сайті.
